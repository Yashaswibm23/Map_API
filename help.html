Directions to use our MAP API Layers

  1. Sign up for an account on the website.
  2. Confirm your email address by checking your inbox.
  3. If you haven't received an email, request a resend of the email verification link.
  4. If you're not logged in, you can only view the demo map on the website for 5 minutes.
  5. After confirming your email address, log in.
  6. You will be redirected to the home page. Click on the "View Map" button.
  7. Access a page with three layers: Nilaya, Roads, and Phoenix.
  8. To access these layers, you need an API key.
  9. Generate an API key by clicking on "GET API KEY" at the top of the page.
  10. Request an API key for the specific layer you want to view (one at a time).
  11. If you want access to two or three layers, click on "GET API KEY" at the top of the webpage, select the layers from the dropdown, and click "GET API KEY" for each layer.
  12. You now have access to the requested layers.
  13. If you are unable to request certain layers, it means you don't have the API key for those specific layers. An error message will be displayed indicating that you don't have permission to view Nilaya, Phoenix, or Roads.

    14. Note: Inconsistencies may occur as the website is still under development.
